USO DE ETIQUETAS EN HTML

Etiqueta html (Documento HTML)

* Es la etiqueta principal. Todo el código HTML debe estar dentro de esta etiqueta.
* Le indica al navegador que está leyendo una página web.

Ejemplo

<html>Toda la informacion se generara aqui</html>

Etiqueta head (Encabezado del documento)

* No se muestra directamente en la página.
* Contiene información técnica: el título de la pestaña, el conjunto de caracteres, enlaces a archivos CSS, scripts, etc.

Ejemplo

<head>
  <title>Mi Página Web</title>
</head>

Etiqueta body (Cuerpo del documento)

* Contiene todo lo que el usuario ve en la página web: texto, imágenes, enlaces, botones, etc.

Ejemplo

<body>
  <h1>Hola Mundo</h1>
  <p>Bienvenido a mi página web.</p>
</body>

Etiquetas de encabezados (h1 hasta h6)

* Se usan para títulos o subtítulos.
* h1 es el más importante (más grande), y h6 el menos importante (más pequeño).

Ejemplo

Encabezado Principal

Subtítulo

Etiqueta P (Párrafo)

* Se usa para escribir bloques de texto normales.
* Se agrega automáticamente un pequeño espacio antes y después del párrafo.

Ejemplo

Es un párrafo donde explicas algo interesante

Etiqueta a (Enlaces)

* Sirve para enlazar a otra página web, archivo o sección de la misma página.
* Necesita el atributo href (dirección del enlace).

Ejemplo

Ir a Google

Etiqueta img (Agrega imagenes)

* Muestra una imagen en la página.
* Necesita el atributo src (ruta de la imagen).

Ejemplo

Etiqueta table (Tablas)

* Se usa para mostrar información en filas y columnas.
* tr (define una fila)
* th (define una celda de encabezado)
* td (define una celda de datos)

Ejemplo

Nombre Edad
Cristhian 30
Andres 25

Etiqueta de video (Reproductor de video)

* Se usa para insertar y reproducir videos directamente en la página web.

Etiqueta audio (Reproductor de audio)

* Se usa para reproducir archivos de audio (música, grabaciones, etc.).

Etiqueta canvas (Área de dibujo)

* Crea un área en blanco donde se pueden dibujar gráficos, animaciones, juegos, etc. mediante JavaScript.

Etiqueta figure (Contenedor de contenido visual)

* Agrupa imágenes, diagramas, gráficos, etc., con su leyenda o descripción.

Etiqueta figcaption (Leyenda de una figura)

* Proporciona una descripción o título para el contenido dentro de <figure>.

Etiqueta progress (Barra de progreso)

* Muestra una barra de progreso (por ejemplo, carga de archivos, tareas completadas).

70%

Etiqueta meter (Indicador de medida)

* Muestra una medida dentro de un rango conocido, como el nivel de batería, porcentaje, temperatura, etc.

Nivel de batería: 60%

Etiqueta details (Contenedor plegable)

* Crea un panel que se puede abrir o cerrar para mostrar más información.

Ver detalles

Este es el contenido oculto que se muestra al hacer clic.

Etiqueta summary (Título del panel)

* Define el título visible del contenedor <details>. Al hacer clic, se despliega el contenido.

Etiqueta mark (Texto resaltado)

Resalta texto como si lo marcaras con un resaltador amarillo. Útil para búsquedas o énfasis.

Esta palabra está resaltada por una razón importante.
